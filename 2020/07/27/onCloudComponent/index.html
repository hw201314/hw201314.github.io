declarationTable

  1. declarationTable 多功能表格,自定义固定行列,可编辑单元格,自动合计,溢出滚动,滚动条样式(默认统一样式),表头事件,表格数据提交

    属性名 类型 默认值 描述
    *dataSource array [] (必填)dataSource表格日数据,根据dayTime匹配24小时数据,格式如下,
    column array [] 自定义固定列,对应每行头title,默认空,开启固定列时,每一列设置dataIndex关联dataSource数据源的dayTime数据作为头title显示,dayTimet其同级参数column为一行数据铺开,可设置参数align,width,fixed自定义样式,格式如下,
    lineList array [“0时”…”23时”] 自定义固定行,对应每列头title,默认24小时,格式如下
    total boolean false 是否开启行值合计数,行为number时适用,自动计算行值 表头添加total开启
    scroll object {x:0,y:0} x:横向滚动，也可用于指定滚动区域的宽和高，可以设置为像素值，百分比 y:纵向滚动
    editTable boolean false 是否开启可编辑单位格,默认不开启
    isClickHeader boolean false 是否开启表头事件
    handleClickHeader handleClickHeader(column){onClick: () =>{}} 表头事件,表头点击回调,待修改,支持获取当前点击元素
    submitTable 父组件自定义事件通过ref获取子组件的dataSource提交

自定义效果 根据年月日电量申报

默认 根据曲线管理
