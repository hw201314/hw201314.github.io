曲线管理

1.曲线管理模块

属性名 类型 默认值 描述
key string 数据标识
month0 string 曲线模块月列标题属性
month1-12 string 曲线模块月列内容属性 对应1-12月
model array 典型日属性(每组数据为一列)
model_title string 典型日标题
modelType1 string 典型日内容属性
season array 季节
season_title string 季节标题
season_1 string 季节内容属性 season_1-season_4 对应1-4季节
month1_dayList array 每月的日期信息集合,类推 month1_dayList-month12_dayList 对应每月日期信息
day0 string 曲线模块日列标题属性
day0-day31 string 对应每日属性 1-31号
day1_type obj 对应1日类型,类推day1_type…day31_type 对应1-31日类型
tradingday number 现货交易日设置类型 0无状态 1为现货交易日
holiday number 节假日设置类型 0工作日 1节假日 2星期六 3星期日
day1_hourList array 每日的小时信息集合 ,day1_hourList-day31_hourList 对应每天24时信息
hour0 string 曲线模块时列标题属性
hour1-hour24 string 对应每小时属性 1- 24小时
